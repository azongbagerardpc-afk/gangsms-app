# 🔑 Récupérer vos clés Supabase en 1 MINUTE

## Étape 1 : Se connecter à Supabase
1. Va sur **https://supabase.com/dashboard**
2. Clique "Continue with GitHub" (ou ton email)
3. Connecte-toi

## Étape 2 : Ouvrir ton projet "gangsms"
- Une fois connecté, tu vois la liste de tes projets
- Clique sur **"gangsms"**

## Étape 3 : Aller dans Settings
- En bas à gauche, clique sur ⚙️ **Settings**

## Étape 4 : Copier les clés dans API
- Clique sur **API** dans le menu de gauche
- Tu vois deux sections :
  - **Project URL** (commence par `https://...`)
  - **anon public** (commence par `eyJ...`)

## Étape 5 : Copier / Coller
```
Copie Project URL   → c'est kSupabaseUrl
Copie anon public   → c'est kSupabaseAnonKey
```

## Étape 6 : Les envoyer ici 📝

Une fois que tu as les 2 clés + le nom de ton repo GitHub → je fais le reste automatiquement ! ✅

**Besoin juste de :**
- `kSupabaseUrl`
- `kSupabaseAnonKey` 
- Nom repo GitHub (ex: gangsms-app)
